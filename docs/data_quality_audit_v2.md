## Semantic Notes

- `planetaryBoundaryLayer_0_VRATE` is NCEP GRIB `VRATE`, ventilation rate in m^2/s, not vertical wind speed. Official table: https://www.nco.ncep.noaa.gov/pmb/docs/grib2/grib2_doc/grib2_table4-2-0-2.shtml
- LDAPS cloud variables are fractions near [0, 1], while GFS cloud variables are percentages near [0, 100]. They must not share a common clipping threshold.
- GFS `surface_0_prate` is a rate while `surface_0_tp` is total precipitation. LDAPS rate and amount fields likewise require separate handling.
- Large spatial robust-z values in cloud and precipitation fields are not automatically errors because zero-heavy localized weather makes MAD very small.
