# 데이터 변수 구조

`data/` 폴더의 각 파일별 변수(컬럼) 구조를 정리한 문서입니다. 원본 설명은 [data_description.md](../data/data_description.md) 참고.

## 파일 목록

| 구분 | 파일 | 행 수 | 내용 |
|---|---|---:|---|
| 학습 | `train/ldaps_train.csv` | 420,864 | 학습 기간 LDAPS 기상 예보 |
| 학습 | `train/gfs_train.csv` | 236,736 | 학습 기간 GFS 기상 예보 |
| 학습 | `train/train_labels.csv` | 26,304 | 학습 기간 KPX 그룹별 실제 발전량 |
| 학습 | `train/scada_vestas_train.csv` | 157,819 | VESTAS 터빈 SCADA 실측 |
| 학습 | `train/scada_unison_train.csv` | 105,264 | UNISON 터빈 SCADA 실측 |
| 평가 | `test/ldaps_test.csv` | 140,160 | 평가 기간 LDAPS 기상 예보 |
| 평가 | `test/gfs_test.csv` | 78,840 | 평가 기간 GFS 기상 예보 |
| 제출 양식 | `sample_submission.csv` | 8,760 | 예측값 제출 양식 |
| 메타 | `info.xlsx` | - | KPX 그룹·터빈·설비용량·위치 정보 |

- CSV 인코딩: `UTF-8 with BOM` (`utf-8-sig`)
- 시간대: 모두 KST, 형식 `YYYY-MM-DD HH:MM:SS`

## 9. 주의사항

- 평가 기간의 실제 발전량, SCADA, 비공개 운영 데이터, 사후 보정자료, 재분석자료는 예측에 사용 불가
