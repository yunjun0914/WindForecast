# 현재 사용 변수 목록

`utils/preprocessing.py`의 `build_weather_features()`가 만들어내는 최종 피처 테이블 기준. 총 **58개 피처**, `kpx_group_1/2/3` 3개 모델(그룹별 RF/LGBM/XGB) 전부 동일한 피처 세트를 사용합니다.

> 원래 101개(기온/습도/기압/복사/구름/강수 등 비바람 변수 포함)였으나, 변수 수가 많아 과적합 우려가 있어 **바람 관련 변수만 남기도록 축소**했습니다. 다만 축소 후 검증 NMAE는 오히려 소폭 나빠졌습니다(그룹별 약 0.3~0.5%p 상승) — 제거된 변수들이 순수 노이즈는 아니었던 것으로 보이나, 일단 바람 변수만 쓰는 것으로 확정했습니다.

## 요약

| 분류 | 개수 |
|---|---:|
| LDAPS 원본(바람) | 8 |
| LDAPS 파생(바람) | 8 |
| GFS 원본(바람) | 16 |
| GFS 파생(바람) | 14 |
| 시간주기 인코딩 | 12 |
| **합계** | **58** |

**제외된 변수(비바람 계열)**: 기온, 이슬점, 상대습도, 비습, 기압(지표/해면), 경계층 높이, 단파/장파 복사, 운량(상/중/하/전체), 강수/강설 관련, 지형/지표 마스크, 지위고도, 수증기압(파생) 등. `docs/데이터 변수 구조.md`에 원본 전체 목록 있음.

현재 이 58개 외에 **시계열 lag/이동평균 등은 포함하지 않음** (의도적으로 제외, tree 계열 + 주기 인코딩으로 대체).
