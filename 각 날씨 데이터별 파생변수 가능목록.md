# 각 날씨 데이터별 파생변수 가능목록

작성일: 2026-07-16 KST  
대상: LDAPS, GFS, NOAA GEFS  
목적: source별 원시 예보에서 만들 수 있는 파생변수와 그 물리적 의미를 정리하고, 이후 ablation에서 한 번에 하나의 정보군만 추가하기 위한 기준표로 사용한다.

## 1. 읽는 법

상태 표기:

| 상태 | 의미 |
|---|---|
| `현재 core` | 현재 source expert 입력에 이미 포함됨 |
| `보유/미실험` | 원시 컬럼이 있어 지금 바로 계산할 수 있지만 새 source expert에서는 아직 검증하지 않음 |
| `실험 후 기각` | 현재 source expert의 matched OOF에서 추가했지만 유지 조건을 충족하지 못함 |
| `추가수집` | 현재 저장 파일에는 필요한 원시 변수가 없음 |

후보 가치 표기:

| 우선순위 | 의미 |
|---|---|
| `P1` | 원시값과 중복이 적고 물리적 의미가 강해 먼저 볼 후보 |
| `P2` | 유용할 가능성이 있지만 다른 변수와 결합하거나 주의 깊은 구현이 필요 |
| `P3` | 신호가 간접적이거나 현재 데이터에서 잡음이 될 가능성이 큼 |

`P1`은 한 번에 모두 넣는 실행 묶음이 아니다. 각 파생변수 자체의 후보 가치를 뜻하며, 실제 투입 순서는 아래 정보 단계로 결정한다.

정보 단계:

| 단계 | 분류 | 새 정보 여부 | 의미 | 투입 조건 |
|---|---|---|---|---|
| `S0` | 현재 core | 기준 | 이미 검증 중인 source별 최소 입력 | 항상 기준으로 유지 |
| `S1` | 원천 신규정보 | 있음 | 기존 core에 없던 source 고유 raw field와 필수 speed | 하나의 source block씩 독립 ablation |
| `S2` | 물리 파생 | 원칙적으로 없음 | 채택된 raw field를 shear, density, spread ratio처럼 더 쉬운 좌표로 변환 | 부모 `S1` block이 유지된 뒤에만 추가 |
| `S3` | 공간 파생 | 없음 | 채택된 공간장의 gradient, coherence, upwind 구조 요약 | raw grid expert가 부모 field를 사용한 뒤 검증 |
| `S4` | 시간 파생 | 없음 | 같은 issue 24시간 궤적의 ramp, peak, envelope 요약 | TCN 외 모델 우선, TCN에는 소수만 검증 |
| `S5` | source 간 메타 | 조합 정보 | 독립 expert의 disagreement, confidence, conditional blend | source별 OOF가 고정된 뒤 검증 |

따라서 `P1-S1`은 실제 신규 예보정보라서 가장 먼저 볼 후보이고, `P1-S2/S3/S4`는 입력 정보량을 늘리기보다 모델이 기존 정보를 읽기 쉽게 만드는 후보이다.

중요 원칙:

- 이 문서는 가능한 후보의 지도이지, 모든 변수를 한 번에 넣으라는 목록이 아니다.
- 새 실험은 `원시 변수 1개` 또는 `하나의 물리적으로 닫힌 파생변수군`만 추가한다.
- `S1`이 실패하면 그 raw field에 의존하는 `S2-S4`를 자동으로 이어서 실험하지 않는다.
- `S2-S4`는 부모 변수가 채택됐거나, 변환 자체가 raw보다 명확한 물리적 이유가 있을 때만 예외적으로 단독 비교한다.
- 같은 정보를 `u/v`, speed, angle, sector, 제곱, 세제곱 형태로 동시에 무더기 복제하지 않는다.
- scaling과 통계량 fitting은 outer-train에서만 수행한다.
- 같은 issue의 24시간 예보는 예측기준시점에 함께 공개되므로 issue 내부의 앞뒤 시간 예보를 사용하는 파생은 허용된다.
- 관측값, same-time AWS, 평가기간 실제 발전량, 사후 재분석값은 포함하지 않는다.

| Source | 공간 구조 | 시간 구조 | 현재 보유한 주요 원시 정보 | 현재 core 입력 |

## 3. 공통 풍속·풍향 파생변수

아래 파생은 대응하는 `u/v`가 있는 모든 source와 높이에 적용할 수 있다.

| 파생변수 | 계산 예시 | 의미 | 권장 상태 | 주의점 |
|---|---|---|---|---|
| 풍속 `ws` | `sqrt(u^2 + v^2)` | 바람 벡터의 크기, 발전량과 가장 직접적인 기상 입력 | `현재 core`, P1 | vector pair마다 정확히 한 번만 생성 |
| 풍향 sin/cos | meteorological direction을 구한 뒤 `sin(theta), cos(theta)` | 0/360도 경계를 끊지 않고 풍향 표현 | `보유/미실험`, P2 | raw `u/v`에 이미 방향 정보가 있어 중복 가능 |
| 단위 방향벡터 | `u/(ws+eps), v/(ws+eps)` | 풍속 크기와 방향을 분리 | `보유/미실험`, P2 | 저풍속에서 불안정하므로 minimum speed mask 필요 |
| 방향 sector | 풍향을 8/12개 sector로 구분 | 지형, wake, 계절풍 방향 regime | `보유/미실험`, P2 | hard bin 경계와 차원 증가에 주의 |
| along-wind / cross-wind | 단지 주축에 `u/v`를 투영 | 단지 배치축을 따라 흐르는 바람과 횡풍 분리 | `보유/미실험`, P1 | 단지 또는 그룹의 물리적 주축 정의 필요 |
| 풍력밀도 단순 proxy | `ws^3` | 풍력의 속도 세제곱 관계를 반영 | `보유/미실험`, P2 | cut-in/rated/cut-out에서 실제 power curve와 다름 |
| 공기밀도 보정 풍력밀도 | `0.5 * rho * ws^3` | 기온·기압·습도에 따른 가용 운동에너지 | `보유/미실험`, P1 | `rho`를 계산할 열역학 변수가 함께 있어야 함 |
| 속도 regime | cut-in/ramp/rated에 대응하는 speed band | power curve의 비선형 구간 표현 | `보유/미실험`, P2 | 임계값은 터빈 모델과 hub height에 따라 다름 |
| 풍향 회전량 | wrapped `theta(t)-theta(t-k)` | 시간에 따른 바람 방향 전환 | `보유/미실험`, P2 | 각도 차는 반드시 circular difference 사용 |
| vector ramp | `sqrt((du)^2 + (dv)^2)` | 속도와 방향 변화가 합쳐진 바람 변화량 | `보유/미실험`, P1 | 단순 `d(ws)`보다 방향 전환도 포착 |

## 4. LDAPS 파생변수

### 4.1 바람과 경계층

사용 가능한 원시 바람:

```text
10m u/v
50m u/v max
50m u/v min
5m boundary-layer X/Y wind
```

| 파생변수 | 원시 변수 / 계산 | 의미 | 상태·우선순위 | 주의점 |
|---|---|---|---|---|
| `ws10` | 10m `u/v` speed | 지표 부근 평균 풍황 | `현재 core`, P1 | 16개 격자 identity 유지 |
| `ws50max` | 50m max `u/v` speed | 50m 강풍 envelope | `현재 core`, P1 | component max가 같은 순간의 vector라는 보장이 없을 수 있음 |
| `ws50min` | 50m min `u/v` speed | 50m 약풍 envelope | `현재 core`, P1 | component min의 물리적 결합에 주의 |
| `ws5bl` | 5m X/Y BL wind speed | 지표 마찰층 바람 | `보유/미실험`, P1 | 현재 plan의 LDAPS L1 후보 |
| 50m envelope 폭 | `ws50max - ws50min` | 시간/모델 내부 풍속 변동성 또는 불확실성 proxy | `보유/미실험`, P1 | 음수가 나오면 원시 max/min 조합 의미 재검증 |
| 50m envelope 비율 | `(ws50max-ws50min)/(0.5*(ws50max+ws50min)+eps)` | 평균 크기에 상대적인 변동폭 | `보유/미실험`, P2 | 저풍속 폭주 방지 clip 필요 |
| 10-5m shear | `log(ws10/ws5)/log(10/5)` | 매우 낮은 고도의 지표 마찰 | `보유/미실험`, P2 | 5m BLWS의 정의가 일반 5m wind와 동일한지 확인 필요 |
| 50-10m shear envelope | max/min 각각의 log shear | hub-height 외삽 범위 | `보유/미실험`, P1 | 50m max/min은 mean profile이 아니므로 true shear가 아니라 envelope |
| 50-10m veer | 50m과 10m wrapped direction 차 | 고도에 따른 풍향 회전 | `보유/미실험`, P2 | max/min vector별로 따로 계산하거나 제외 |
| hub-height speed 범위 | shear를 이용해 turbine hub height로 외삽 | 실제 rotor 높이의 유효 풍속 범위 | `보유/미실험`, P2 | hub-height metadata와 shear 안정화 필요 |
| rotor-equivalent wind proxy | hub/rotor 상·하단 속도를 shear로 외삽해 평균 | rotor 전체가 받는 바람 | `보유/미실험`, P2 | 50m 이하 profile만으로 높은 hub를 외삽하므로 불확실성 큼 |
| BLH 대비 hub 비율 | `hub_height/max(BLH,20m)` | rotor가 경계층 내부/상단 어디에 놓이는지 | `실험/미채택`, P1 | `117/max(BLH,20)` 1채널 OOF에서 standalone `-0.002539`, final blend `-0.000044` |
| low-BLH flag | `BLH < hub_height` | 안정 경계층, 강한 shear 가능 regime | `보유/미실험`, P2 | hard flag보다 smooth ratio가 우선 |

### 4.2 열역학과 기압

사용 가능한 원시 변수:

```text
2m temperature, dew point, relative humidity, specific humidity
surface pressure, mean-sea-level pressure
boundary-layer height
```

| 파생변수 | 계산 | 의미 | 상태·우선순위 | 주의점 |
|---|---|---|---|---|
| 섭씨/절대온도 정규화 | `T_C=T_K-273.15` | 해석 가능한 온도 표현 | `보유/미실험`, P3 | 모델에는 원시 K를 scale해도 정보는 동일 |
| dew-point depression | `T - Td` | 공기 포화도와 안개/구름 가능성 | `보유/미실험`, P2 | T와 Td 단위 통일 필요 |
| 수증기압 | T와 RH 또는 Td로 계산 | 습윤 공기의 열역학 상태 | `보유/미실험`, P2 | RH/q/Td를 모두 동시에 넣으면 중복 큼 |
| virtual temperature | `Tv=T*(1+0.61q)` | 수증기를 고려한 유효 온도 | `보유/미실험`, P1 | q 단위가 kg/kg인지 audit 필요 |
| air density `rho` | `sp/(R*Tv)` | 같은 풍속에서 발전 가능한 에너지 밀도 | `보유/미실험`, P1 | pressure 단독은 OOF에서 기각됐으며 T/q와 닫힌 조합으로 봐야 함 |
| density-normalized speed | `ws*(rho/rho0)^(1/3)` | 표준 공기밀도 기준 등가풍속 | `보유/미실험`, P1 | turbine power curve와 잘 맞는 물리 변환 후보 |
| surface-MSL pressure gap | `prmsl-sp` | 지형 고도와 기단 상태가 섞인 hydrostatic proxy | `보유/미실험`, P2 | 대부분 정적 고도 신호일 수 있음 |
| pressure anomaly | 각 시각 grid pressure에서 장기/공간 평균 제거 | 기압계의 상대적 강도 | `보유/미실험`, P2 | 장기 평균은 outer-train에서만 fit |
| pressure tendency | same-issue `+/-3h` slope | 저기압/전선 접근과 풍황 변화 | `LDAPS 실험/미채택`, P1 | standalone `-0.002667`, final blend `-0.001403`; GFS는 미실험 |
| pressure gradient | 격자 좌표에 대한 동서/남북 slope | 기압경도력과 풍장 강화 가능성 | `보유/미실험`, P1 | 16개 고해상도 격자에서 유리, 지형 영향 분리 필요 |
| BLH tendency | `BLH(t)-BLH(t-k)` | 경계층 성장·붕괴 | `보유/미실험`, P2 | 일사 주기와 강하게 중복 |

현재 실험 기록:

- `surface_0_sp` 한 채널만 추가한 LDAPS-SP는 standalone `0.622694 -> 0.621934`로 기각됐다.
- `meanSea_0_prmsl` 한 채널은 standalone `0.622694 -> 0.622073`, final blend `0.630926 -> 0.629152`로 기각됐다.
- surface pressure, 3h tendency, MSLP가 모두 실패했으므로 pressure scalar 직접입력 계열은 종료한다.
- 다음에 density를 볼 경우 T/q/sp를 함께 사용한 하나의 물리 파생량으로 검증해야 한다.
- Thermo/PBL raw family `T/Td/RH/q/BLH`는 standalone `-0.002378`, final blend `+0.000095`였으나 2023과 g3 하락으로 미채택했다.
- radiation/cloud/precipitation/snow/terrain raw family는 standalone `-0.004553`, final blend `-0.002019`로 기각했다.
- 상수 `surface_0_lsm`은 제외했으며, 이 결과로 LDAPS raw family 직접입력 점검은 종료했다.

### 4.3 복사·운량·강수·지표

| 파생변수 | 원시 변수 / 계산 | 의미 | 상태·우선순위 | 주의점 |
|---|---|---|---|---|

### 6.1 Ensemble mean 풍장

```text

10m, 925hPa, 850hPa, 700hPa u/v spread
```

```text
gust mean
gust spread
```

| 파생변수 | 계산 | 의미 | 상태·우선순위 | 주의점 |
|---|---|---|---|---|
| mean speed at 10/925/850 | mean `u/v`에서 speed | ensemble mean 풍장 | `현재 core`, P1 | `mean(speed)`가 아니라 `speed(mean vector)`임 |
| mean speed at 700 | 700hPa mean `u/v` speed | 더 높은 synoptic steering flow | `실험/미채택`, P1 | standalone `-0.000826`, final blend `-0.001366` |
| mean direction sin/cos | mean vector direction | ensemble 평균 풍향 | `보유/미실험`, P2 | raw mean `u/v`와 중복 가능 |
| vertical speed coupling | 925-10, 850-925, 700-850 ratio/difference | 상층 풍장이 지표로 전달되는 정도 | `보유/미실험`, P1 | pressure level 간 값은 true geometric shear가 아님 |
| vertical direction coherence | level별 mean unit vector의 합성 norm | 고도별 바람 정렬 정도 | `보유/미실험`, P1 | front/veer regime 표현 |
| mean gust factor | `gust_mean/(ws10_mean+eps)` | 평균 gustiness proxy | `보유/미실험`, P1 | 서로 다른 0.25/0.5도 grid를 latent에서 결합 권장 |
| synoptic-surface decoupling | 850/925 wind와 10m wind 차이 | 안정 경계층 또는 지형 차폐 | `보유/미실험`, P1 | 공간 해상도 차이 처리 필요 |

### 6.2 Ensemble spread와 불확실성

| 파생변수 | 계산 | 의미 | 상태·우선순위 | 주의점 |
|---|---|---|---|---|
| component spread norm | `sqrt(u_sprd^2+v_sprd^2)` | 바람 성분 예측의 전체 불확실성 크기 proxy | `실험/미채택`, P1 | Near final `-0.001751`, Upper final `-0.001841`; 실제 speed 표준편차와 동일하지 않음 |
| relative wind spread | `spread_norm/(mean_ws+eps)` | 평균풍 대비 상대적 불확실성 | `보유/미실험`, P1 | 저풍속 clip 필요 |
| gust spread | 원시 `gust_sprd` | 돌풍 예측의 ensemble 불확실성 | `실험/미채택`, P1 | Near family에서 10/925 spread와 함께 넣었으나 final `-0.001751` |
| relative gust spread | `gust_sprd/(gust_mean+eps)` | 돌풍 강도 대비 불확실성 | `보유/미실험`, P1 | low-gust 안정화 필요 |
| uncertainty-adjusted wind | `mean_ws +/- k*spread_proxy` | 보수적/공격적 풍속 범위 | `보유/미실험`, P2 | k sweep는 별도 승인 없는 튜닝이 되므로 고정 근거 필요 |
| mean-spread interaction | `mean_ws * relative_spread` | 강풍이면서 불확실한 regime | `보유/미실험`, P2 | raw 둘을 모델이 비선형 결합할 수 있으면 중복 가능 |
| spread vertical gradient | level별 spread 차이/비율 | 어느 고도에서 예보가 불안정한지 | `보유/미실험`, P2 | pressure level별 representativeness 차이 |
| spread ramp | spread의 1/3/6h 변화 | 예보 불확실성이 커지는 시점 | `보유/미실험`, P2 | 원래 3시간 자료 보간의 인공 smoothness 고려 |
| spatial uncertainty concentration | spread grid max/mean/std와 위치 | 불확실성이 특정 방향/영역에 집중되는지 | `보유/미실험`, P2 | summary를 과도하게 늘리지 않음 |
| cut-in/rated 도달 확률 proxy | mean과 spread에 정규분포를 가정해 threshold 확률 계산 | 발전 regime 진입 확률 | `보유/미실험`, P2 | raw member와 u-v covariance가 없어 근사치일 뿐 |
| source confidence | `1/(spread+eps)` 또는 bounded transform | GEFS 예측을 얼마나 신뢰할지 | `보유/미실험`, P1 | source blend gate에 쓰려면 OOF 검증 필요 |

현재 GEFS는 ensemble member별 값이 아니라 mean/spread만 저장한다. 따라서 다음은 정확히 만들 수 없다.

- 진짜 member-wise speed/power 분포
- empirical quantile
- member별 cut-in/rated 확률
- u-v covariance를 반영한 정확한 풍향 불확실성

### 6.3 GEFS에서 현재 불가능하거나 추가수집이 필요한 변수

| 후보 | 현재 상태 | 필요한 추가 원시 자료 | 의미 |
|---|---|---|---|
| surface pressure / MSLP | `추가수집` | GEFS surface pressure 또는 PRMSL | 기압계, 공기밀도 |
| 2m temperature/humidity | `추가수집` | GEFS 2m T, Td/RH/q | ensemble air-density uncertainty |
| PBL height | `추가수집` | GEFS PBLH | 경계층 안정·혼합 |
| cloud/radiation/precipitation | `추가수집` | 해당 GEFS mean/spread 또는 members | weather regime uncertainty |

추가수집 변수는 공개시각 metadata, 라이선스, 재현 가능한 다운로드 코드, unsafe run fallback을 함께 갱신해야 한다.

## 7. 모든 source에 적용 가능한 공간장 파생변수

공간 통계는 각 grid 위치를 지우지 않는 것이 우선이다. source expert가 raw grid를 직접 읽는 경우 아래 통계는 보조 채널 또는 진단값으로 소수만 추가한다.

| 파생변수 | 계산 | 의미 | 권장 우선순위 | 주의점 |
|---|---|---|---|---|
| vector spatial mean | grid `u/v` 각각 평균 | 영역 평균 풍장 | P2 | speed를 먼저 평균하는 것과 다름 |
| speed spatial mean/std | grid speed 평균과 표준편차 | 평균 풍속과 공간 비균질성 | P1 | raw grid encoder가 이미 학습할 수 있는 정보 |
| min/max/range | grid speed extrema | 국지 강풍·약풍 범위 | P2 | outlier에 민감 |
| robust spread | p90-p10 또는 MAD | outlier에 덜 민감한 공간 변동성 | P1 | quantile을 여러 개 만들지 않음 |
| vector coherence | `norm(mean(u,v))/mean(ws)` | grid 간 풍향 일치도 | P1 | 0은 방향 불일치, 1은 정렬 |
| circular direction spread | 방향 resultant length 기반 | 공간적으로 풍향이 얼마나 퍼지는지 | P2 | 저풍속 grid 제외 필요 |
| center-minus-ring | 중심/최근접 grid 값 - 외곽 평균 | 국지 효과와 광역 배경 차이 | P1 | source별 중심 정의 필요 |
| east-west / north-south slope | 좌표에 대한 선형회귀 slope | 풍속·기압·온도 공간 gradient | P1 | 단순 row/column 차보다 실제 좌표 사용 |
| gradient magnitude/direction | `sqrt(gx^2+gy^2)`, `atan2(gy,gx)` | front/jet의 방향과 강도 | P2 | gradient angle은 sin/cos 권장 |
| Laplacian/curvature | 주변값 - 중심값 | 국지 최대·최소와 굽음 | P2 | LDAPS irregular mask와 edge 처리 필요 |
| argmax centroid | 큰 값의 가중 중심 좌표 | 강풍 core가 어느 쪽에 있는지 | P2 | hard argmax보다 soft weighted centroid 권장 |
| divergence proxy | `du/dx + dv/dy` | 수평 수렴·발산 | P2 | 작은 grid에서 미분 noise 큼 |
| vorticity proxy | `dv/dx - du/dy` | 회전성 흐름 | P2 | 좌표 간격 보정 필수 |
| deformation | stretching/shearing deformation | 풍장 변형과 front 구조 | P3 | 현재 grid 수에서 불안정할 수 있음 |
| scalar advection | `u*dS/dx + v*dS/dy` | 기압·온도·습도장이 바람에 의해 이동하는 정도 | P2 | 동일 높이/해상도의 wind와 scalar를 사용 |
| upwind weighted mean | 예보 풍향의 상류 grid에 높은 가중 | 다가오는 공기 흐름 | P1 | target 시각 풍향은 예보값이므로 규칙상 사용 가능 |
| downwind-minus-upwind | 상·하류 풍속/기압 차이 | 지형 차폐와 흐름 변화 proxy | P2 | 단지 위치와 grid geometry 필요 |
| directional terrain interaction | 풍향 x elevation slope/aspect | 특정 풍향에서 지형 가속·차폐 | P1 | 정적 terrain과 예보 풍향의 설명 가능한 결합 |

## 8. 모든 source에 적용 가능한 24시간 궤적 파생변수

한 issue의 24개 예보는 같은 `data_available_kst_dtm`에 함께 공개된다. 따라서 아래 연산은 미래 관측 누수가 아니라 예보 궤적 요약이다.

| 파생변수 | 계산 예시 | 의미 | 권장 우선순위 | 주의점 |
|---|---|---|---|---|
| lead | 12-35h | 예보 horizon에 따른 bias/불확실성 | `현재 core`, P1 | 현재 time feature에 포함 |
| 1/3/6h difference | `x(t)-x(t-k)` | 단기 ramp | P1 | issue 경계 밖으로 넘어가지 않음 |
| centered difference | `(x(t+1)-x(t-1))/2` | 같은 issue 내 국소 기울기 | P1 | full-context 모델에서만 사용 |
| acceleration | `diff(diff(x))` | ramp가 빨라지거나 느려지는 정도 | P2 | noise 확대 가능 |
| rolling mean/std | 3/6h window | 국소 평균과 변동성 | P2 | window 여러 개 무한복제 금지 |
| local min/max/range | 주변 +/-2~3h | target 시각 주변 예보 envelope | P1 | current target을 포함한 forecast-only 연산 |
| issue mean/std/p10/p90 | 24시간 전체 요약 | 하루 풍황 regime | P1 | 시간 위치 정보는 원시 궤적과 함께 유지 |
| daily energy proxy | `sum(ws^3)` | 하루 전체 가용 풍력 수준 | P2 | 발전량 합이 아니라 weather proxy |
| max ramp와 발생 lead | `max(abs(diff(ws)))` | 가장 큰 풍황 전환의 크기와 시점 | P1 | argmax lead는 categorical/cyclic 처리 고려 |
| peak wind와 발생 lead | issue max 및 위치 | 강풍 peak와 시점 | P1 | max 하나만으로 궤적을 대체하지 않음 |
| threshold duration | speed regime에 머문 시간 | cut-in/rated 구간 지속성 | P2 | turbine별 threshold 근거 필요 |
| threshold crossing count | regime 경계 통과 횟수 | 불안정한 출력 전환 가능성 | P2 | 보간 noise에 민감 |
| direction turn rate | wrapped angle difference | 풍향 급변 | P2 | 저풍속 시각 mask 필요 |
| vector-path length | `sum(norm(delta(u,v)))` | 24시간 바람 벡터 변화 총량 | P1 | speed ramp와 다른 정보 |
| temporal coherence | lag correlation 또는 smoothness | 예보 궤적의 지속성 | P2 | 24 point에서 복잡한 spectral feature는 과적합 가능 |
| diurnal phase interaction | HOD sin/cos x wind/BLH/radiation | 일주기 경계층 변화 | P2 | HOD 자체는 현재 core에 포함 |

TCN은 이들 중 많은 국소 패턴을 convolution으로 암묵적으로 학습할 수 있다. 명시적 파생은 TCN이 놓칠 물리량이거나 TREE/PINN처럼 temporal 처리가 약한 모델에 우선 적용한다.

## 9. Source 간 파생변수

source 전문가를 먼저 독립 학습한 뒤 사용할 수 있는 정보다. 서로 다른 해상도의 raw grid를 cell-by-cell 바로 빼지 않는다.

| 파생변수 | 계산 | 의미 | 주의점 |
|---|---|---|---|
| source speed disagreement | source별 farm-centric speed 차이 | 예보원 간 불확실성 | 높이와 공간대표성을 먼저 맞춤 |
| source direction disagreement | circular direction difference | 예보원 간 풍향 불일치 | angle wrap 처리 |
| source prediction spread | 세 expert 발전량 예측의 std/range | model-level uncertainty | 현재 convex blend의 diversity와 직접 연결 |
| source consensus | normalized expert 평균 또는 median | 공통 신호 | weight는 OOF에서만 선택 |
| rank/order | 어느 source가 가장 강한 wind를 예측하는지 | regime-dependent source bias | hard rank보다 smooth difference 우선 |
| lead-conditioned disagreement | horizon별 source 차이 | source별 horizon 강약 | 여러 year에서 반복될 때만 interaction 허용 |
| GEFS-confidence gating | GEFS spread에 따라 GEFS weight 조절 | 불확실할 때 source weight 축소 | spread OOF가 먼저 필요 |
| local-vs-synoptic contrast | LDAPS 국지풍 - GFS/GEFS 광역풍 | 지형·국지 가속 또는 차폐 | 단순 raw 평균보다 source latent/prediction 차이가 안전 |
| cross-source pressure difference | LDAPS `sp/prmsl` - GFS 대응값 | 모델별 기압 bias | GEFS scalar pressure는 현재 없음 |

현재 core source convex blend OOF는 `0.630831`이며 source residual correlation은 약 `0.742-0.827`이다. 따라서 source 간 차이는 실제 diversity를 갖지만, meta weight의 완전한 unbiased 평가는 base model까지 다시 접은 2-level nested CV가 필요하다.

## 10. 단계별 투입 순서와 ablation 단위

### 10.1 S1: 진짜 신규 예보정보

`S1`에서는 interaction이나 요약 통계를 붙이지 않는다. raw vector는 `u/v`와 필수 speed까지만 제공하며, 아래 행을 하나씩 독립적으로 검증한다.

| 순서 | Source | S1 block | 추가 채널 | 기존 대비 | 새로 들어오는 정보 |
|---:|---|---|---:|---:|---|
| 1 | GEFS | core-level ensemble spread | 10/925/850hPa `u/v spread` 6개 + `gust spread` 1개 | combined `10 -> 17` | ensemble 불확실성 |
| 2 | GFS | 10m wind | 10m `u/v/speed` 3개 | `7 -> 10` | 지표 마찰층 풍장 |
| 3 | LDAPS | 5m BL wind | 5m `X/Y/speed` 3개 | `9 -> 12` | 고해상도 지표 마찰층 풍장 |
| 4 | GEFS | 700hPa mean wind | 700hPa `u/v/speed` 3개 | pressure `9 -> 12` | 상층 synoptic steering flow |
| 5 | GFS | vertical wind family | PBL/850/700/500hPa `u/v/speed` + VRATE 13개 | accepted GFS10 `10 -> 23` | `실험/채택`; synoptic wind와 지표풍 연결 재료 |

1-3번을 모두 채택한다고 가정해도 source channel 합은 `26 -> 39`다. 이전의 `26 -> 41` 계산은 GFS의 `100-10m shear`와 `gust factor`라는 `S2` 파생 2개까지 섞은 값이므로, 원천 신규정보 증가량으로는 `39`가 정확하다.

GEFS raw spread S1 결과:

```text
GEFS mean core         = 0.610921
GEFS mean + raw spread = 0.608686
delta                  = -0.002235
```

10/925/850hPa `u/v spread` 6개와 `gust spread` 1개만 추가한 matched standalone OOF에서는 NMAE가 크게 악화됐다. component norm, relative spread, confidence gating은 이 실험에 포함하지 않았으며 자동으로 이어서 실행하지 않는다.

채택 여부는 standalone이 아니라 최종 source ensemble로 판정했다.

```text
LDAPS + GFS + GEFS mean baseline = 0.630831
GEFS spread expert replacement  = 0.630390
delta                           = -0.000441
```

앙상블에서도 g1/g2는 `+0.001631/+0.000124`였지만 g3가 `-0.003079` 하락했다. 따라서 raw spread 7채널 직접입력은 현재 pipeline에 채택하지 않는다. 다만 spread 정보 자체는 폐기하지 않고 component norm, relative spread, confidence gating 같은 파생변수 후보로 보류한다.

GFS 10m wind S1 결과:

```text
GFS standalone core       = 0.610861
GFS standalone + 10m wind = 0.612026  (+0.001165)

core source blend         = 0.630831
GFS 10m expert replacement= 0.630926  (+0.000095)
```

10m `u/v/speed` 3채널은 GFS standalone을 개선했고 최종 ensemble 증가는 작았다. 사용자 결정으로 source expert baseline에 채택했으며, 이후 ensemble 비교 기준은 `LDAPS core + GFS 10m core + GEFS mean core = 0.630926`이다. 동시에 100-10m shear와 gust factor를 만들 수 있는 부모 원천정보로 유지한다.

LDAPS 5m BL wind S1 결과:

```text
LDAPS standalone core      = 0.622694
LDAPS standalone + 5m wind= 0.623029  (+0.000335)

GFS 10m accepted blend    = 0.630926
LDAPS 5m expert replacement=0.630987  (+0.000061)
```

5m `X/Y/speed` 3채널은 g2를 `+0.003912` 올렸지만 g3를 `-0.003619` 낮췄고, held-out year별 방향도 불안정했다. 최종 이득은 noise 수준이므로 사용자 결정 전 보류한다.

### 10.2 S2: 채택된 원천정보의 물리 파생

부모 `S1` block이 standalone 또는 source blend 유지 조건을 통과한 경우에만 다음 행으로 진행한다.

| 부모 block | S2 최소 단위 | 권장 추가량 | 목적 |
|---|---|---:|---|
| GFS 10m | 100-10m shear + gust factor | 2 | 지표 마찰과 gustiness를 직접 표현 |
| LDAPS 5m | 10-5m shear | 1 | 초저층 마찰을 선형적으로 읽기 쉽게 변환 |
| GEFS spread | level별 spread norm + relative spread, relative gust spread | 7 | component uncertainty를 크기와 상대 불확실성으로 변환 |
| GEFS 700hPa | 700-850 coupling + vertical coherence | 2-3 | 수직 풍장 연결과 정렬도 표현 |
| GFS 850hPa | 850-100 coupling + veer | 2-3 | 상층풍 전달과 방향 회전 표현 |
| LDAPS/GFS thermo raw block | `rho` + density-equivalent wind | 2 | T/q/sp를 하나의 닫힌 풍력 물리량으로 축약 |

`rho` 실험은 pressure, temperature, humidity를 각각 잡다한 raw channel로 늘리는 실험이 아니다. 필요한 raw를 계산에만 쓰고, 우선 `rho`와 density-equivalent wind 두 파생량만 모델에 제공한다.

LDAPS hub/BLH ratio P1 결과:

```text
LDAPS standalone core       = 0.622694
LDAPS + 117/max(BLH,20m)    = 0.620155  (-0.002539)

GFS 10m accepted blend      = 0.630926
LDAPS BLH expert replacement= 0.630882  (-0.000044)
```

raw BLH 없이 ratio 한 채널만 추가했지만 standalone FiCR이 하락했고, 최종 blend에서도 g2 `+0.003512`와 g1/g3 `-0.002625/-0.001019`가 교환됐다. 직접입력은 미채택하며 flag/tendency/다른 BLH transform으로 자동 확장하지 않는다.

LDAPS 3h pressure tendency P1 결과:

```text
LDAPS standalone core          = 0.622694
LDAPS + 3h pressure slope      = 0.620027  (-0.002667)

GFS 10m accepted blend         = 0.630926
LDAPS tendency expert replacement=0.629523 (-0.001403)
```

raw pressure 없이 same-issue `+/-3h` Pa/hour slope 한 채널만 추가했지만 NMAE와 FiCR이 모두 악화됐다. tendency 직접입력은 기각하며, spatial pressure gradient나 anomaly는 별도 후보로 남기되 자동 실행하지 않는다.

LDAPS mean sea-level pressure P1 결과:

```text
LDAPS standalone core       = 0.622694
LDAPS + raw MSLP            = 0.622073  (-0.000621)

GFS 10m accepted blend      = 0.630926
LDAPS MSLP expert replacement=0.629152 (-0.001774)
```

MSLP는 surface pressure의 지형 차이를 제거했지만 final blend의 g1/g3를 `-0.002963/-0.003719` 낮췄다. pressure scalar 직접입력은 종료하며, pressure를 T/q와 함께 닫힌 air-density 물리량으로 쓰는 후보만 별도로 남긴다.

### 10.3 S3: 공간 파생

공간 파생은 source의 모든 변수에 복제하지 않는다. `S1/S2`에서 살아남은 대표 풍장 하나 또는 두 개에만 적용한다.

| 최소 block | 최대 추가량/source | 의미 |
|---|---:|---|
| robust spatial spread + vector coherence | 2 | 공간 비균질성과 풍향 정렬도 |
| E-W/N-S slope | 2 | 공간 gradient |
| upwind weighted mean | 1 | 단지로 들어오는 상류 풍황 |

첫 공간 실험은 위 세 행 중 하나만 선택한다. source 하나에 공간요약 5개를 한 번에 넣지 않는다.

### 10.4 S4: 시간 파생

시간 파생도 모든 raw channel에 복제하지 않는다. 대표 speed 또는 vector 하나에만 적용한다.

| 모델군 | 첫 최소 block | 추가량 | 판단 |
|---|---|---:|---|
| TREE/PINN | 1/3/6h vector ramp | 3 | temporal 처리가 약하므로 우선 가치가 큼 |
| TREE/PINN | issue max ramp + peak lead | 2 | 하루 궤적의 급변과 위치 |
| TCN source expert | 명시적 파생 없음 | 0 | 우선 raw 24시간 궤적을 convolution으로 학습 |
| TCN source expert 후속 | vector ramp 1개 | 1 | TCN이 놓치는지 별도 ablation할 때만 추가 |

### 10.5 S5: source 간 메타 파생

세 source expert의 OOF가 고정된 뒤에만 `expert prediction spread`, `source disagreement`, `GEFS confidence gating`을 하나씩 검증한다. 이 단계는 weather tensor 채널을 늘리는 작업이 아니라 최종 ensemble 입력을 늘리는 작업이다.

### 10.6 보류 정보

cloud, radiation, precipitation, snow, 복잡한 vorticity/deformation, 다수 threshold/sector 파생은 `S1-S5`의 직접 후보가 소진되기 전까지 보류한다.

이미 확인된 항목:

| 항목 | 결과 | 판단 |
|---|---:|---|
| LDAPS/GFS raw `surface_0_sp` 추가 | source blend `0.630831 -> 0.630304` | 기각 |
| source별 raw core 독립모델 | LDAPS `0.622694`, GFS `0.610861`, GEFS `0.610921` | 단독보다 source 결합 가치가 큼 |
| meta-year source convex blend | `0.630831`, LDAPS 대비 `+0.008137` | source diversity 신호 확인 |

## 11. 구현 전 필수 점검

| 점검 | 이유 |
|---|---|
| 원시 단위 확인 | K/C, Pa/hPa, %/fraction, kg/kg 혼동 방지 |
| 누적량 정의 확인 | `tp`, 강수/눈 변수의 issue reset과 시간 차분 오류 방지 |
| vector pair 의미 확인 | LDAPS 50m max/min U/V가 동일 event의 vector인지 확인 |
| 저풍속 안정화 | direction, ratio, shear, gust factor 폭주 방지 |
| circular 연산 | 풍향 359도와 1도를 358도 차이로 처리하지 않기 |
| 실제 좌표 사용 | source별 grid 간격과 mask가 다르므로 index 차분 금지 |
| train-only fitting | climatology, anomaly, scaler, threshold를 validation/test에서 학습하지 않기 |
| 중복 제거 | raw와 alias, 물리식 복제본이 동일 값을 반복하지 않게 하기 |
| 결측 provenance | LDAPS test 일부 변수의 issue 내부 보간과 missing audit 유지 |
| GEFS publication audit | unsafe run fallback과 공개시각 검증 유지 |
| 동일 OOF 계약 | outer predictions를 이어 붙인 뒤 group별 metric을 한 번만 계산 |
| 한 실험 한 변화 | 실패 후 임의 feature subset, weight, seed, loss 변경 금지 |

## 12. 핵심 요약

각 source의 가장 큰 강점은 다르다.

| Source | 가장 고유한 정보 | 우선적으로 살릴 파생 |
|---|---|---|
| LDAPS | 1.5 km 국지 공간장과 50m max/min envelope | 지표마찰, envelope 변동성, terrain/upwind spatial feature |
| GFS | 80/100m near-hub wind와 850/700/500hPa 수직 구조 | shear, veer, gust factor, surface-synoptic coupling |
| GEFS | ensemble mean과 spread | uncertainty, confidence, vertical spread coupling |

가장 중요한 방향은 피처 개수를 늘리는 것이 아니라, 각 source만이 가진 고유 정보가 모델 출력과 source blend에 실제로 기여하는지 한 단위씩 검증하는 것이다.
